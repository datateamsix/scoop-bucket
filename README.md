# Scoop Bucket for Email Sentinel

Official Scoop bucket for [Email Sentinel](https://github.com/datateamsix/email-sentinel).

## Installation

```powershell
scoop bucket add datateamsix https://github.com/datateamsix/scoop-bucket
scoop install email-sentinel
``` 

## Available Apps

- **email-sentinel** - Lightweight, smart real-time notification management system

## Updating

```powershell
scoop update
scoop update email-sentinel
```
